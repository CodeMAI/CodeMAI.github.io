1、随机数

1.1、Random

  Random类专门用于生产一个伪随机数，它有两个构造器：一个构造器使用默认(当前时间)的种子，另一个需显示传入一个long型的整数种子。

1.2、ThreadLocalRandom

  ThreadLocalRandom类是Java7中新增类，它是Random类的增强版，在并发访问的环境下，使用它来代替Random可减少多线程资源竞争，保证系统具有较好的性能.，其用法与Random基本一样。
